## ⭐️About Project

**该项目为语音识别系统，将采集完后数据进行傅里叶分析，然后通过特征点捕获实现语音识别。** 
<div>
 
***语言信号的采集*** 通过STM32H750的ADC以高速率进行实时检测，检测到有语音信号后另一个ADC以较低采样率，对语音信号进行采集。
<div>

***语音信号的处理：*** 保证采集到一个完整语素的前提下，进行傅里叶分析。
<div>

***识别：*** 采用相关性分析算法和特征匹配两种方法。
<div>
